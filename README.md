## Возможности

### Подключение
- Автоматическое определение COM портов
- Стабильное USB соединение со станком
- Визуальная индикация состояния подключения
- Автоматическое определение прошивки Marlin

### Ручное управление
- Перемещение по осям X, Y, Z с настраиваемым шагом (0.1, 1, 10, 100 мм)
- Поддержка относительного (G91) и абсолютного (G90) позиционирования
- Установка текущей позиции (G92)
- Парковка инструмента в безопасную позицию
- Возврат в домашнюю позицию (G28)

### Работа с G-кодом
- Загрузка файлов G-кода (.gcode, .nc, .ngc)
- Потоковая передача команд на станок
- Контроль прогресса выполнения
- Экстренная остановка выполнения
- Просмотр содержимого файла перед запуском

### SD карта
- Загрузка файлов G-кода на SD карту станка
- Запуск печати с SD карты
- Просмотр списка файлов на карте
- Прогресс-бар при загрузке файлов

### Визуализация
- Интерактивная рабочая область с координатной сеткой
- Отображение текущей позиции инструмента в реальном времени
- Настраиваемые размеры рабочей области (по умолчанию 350×400×70 мм)
- Цветовая индикация положения по осям

### Безопасность
- Кнопка аварийной остановки (M112 + M410)
- Подтверждение действий при отключении
- Визуальные предупреждения
- Логирование всех операций

### Терминал
- Встроенный терминал для прямого общения со станком
- История отправленных команд
- Цветовая индикация ответов от станка
- Отправка произвольных G-код команд

### Конфигурация
- Сохранение настроек в JSON файлы
- Настройка параметров рабочей области
- Конфигурация шагов двигателей
- Карта высот стола
- Настройка скоростей и ускорений

## Требования

### Аппаратные
- ЧПУ станок с прошивкой Marlin 2.1.2.5
- USB соединение
- SD карта в станке (опционально)

### Программные
- Python 3.7 или выше
- PyQt5 5.15 или выше
- pyserial 3.5 или выше
- Операционная система: Windows 7/10/11, Linux, macOS
